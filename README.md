# DotNet10PreviewLab

このリポジトリは、.NET 10のプレビュー機能を検証するためのコードを集めたものです。以下の内容を含んでいます。

## プロジェクトの目的

このプロジェクトは、.NET 10で導入される新機能や改善点を試し、学習するためのサンプルコードを提供します。特に、以下の機能に焦点を当てています。

- nameof 演算子でバインドされていないジェネリック型
- 新しいコレクションAPI
- その他のC#言語機能のプレビュー

## ファイル構成

- `Program.cs`: メインのエントリーポイント。基本的なサンプルコードが含まれています。
- `OrderedDictionaryTryAdd.cs`: 新しいコレクションAPIの一部であるOrderedDictionaryのサンプルコード。
- `Dockerfile`: プロジェクトをDockerコンテナで実行するための設定ファイル。
